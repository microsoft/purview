# Security Copilot Workshop for Purview

A virtual workshop and hands-on lab guide for Microsoft Security Copilot with Microsoft Purview integration. Includes presentation slides covering Security Copilot capabilities for Purview scenarios and a step-by-step lab guide for hands-on training.

## Products

- Microsoft Security Copilot
- Microsoft Purview

## Prerequisites

- Microsoft Security Copilot access
- Microsoft Purview license (E5 Compliance or equivalent)
- Azure subscription with Security Copilot capacity units

## Getting Started

1. Review the workshop slides (`Security Copilot Virtual Workshop - Slides.pdf`) for an overview of Security Copilot + Purview capabilities.
2. Follow the Lab Guide (`Security_Copilot_Training_LabGuide_ Purview.pdf`) for hands-on exercises.

## Contents

| File | Description |
|------|-------------|
| `Security Copilot Virtual Workshop - Slides.pdf` | 53-slide presentation covering Security Copilot capabilities for Purview |
| `Security_Copilot_Training_LabGuide_ Purview.pdf` | 44-page hands-on lab guide with step-by-step exercises |
